Development / Design

pop!

2019

pop! is an art site for screen printing. It is a concept site for now. This was my final project in the Front-End Web Developer program I recently completed. My goal was to make a responsive site that allowed users to buy and sell art.

My Process

I started with some research. I looked at similar sites and brainstormed on features and what exactly my site was going to do. I sketched out the site UI based on how I wanted it to work on mobile and then desktop. And created a UI prototype with wireframes.

I decided on a look and style for the site including typography, colors, images and icons.

I took a mobile first approach. Using a combination of flexBox and grid to achieve different mobile, tablet, and desktop views. I made everything somewhat flexible and then added breakpoints using media queries to control the design based on the width of the viewers screen.
