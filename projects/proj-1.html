Development / Design

pop!

2019

pop! is an art site for screen printing. This was my final project in the Front-End Web Developer program I recently completed. It is a concept site for now. I included it here because it gives me a chance to talk about my UI and Graphic Design skills. Stay tuned because I intend to bring it to reality.

I started with some research. I looked at similar sites and brainstormed on features and what exactly the site was going to do. I sketched out the site UI based on how I wanted it to work on mobile and then desktop. And created a UI prototype with wireframes.

I used a combination of flexBox and grid to achieve different mobile, tablet, and desktop views. I made everything somewhat flexible and then added breakpoints using media queries to control the design as the view width increases. See it in action.
